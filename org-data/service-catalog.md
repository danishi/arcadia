## 5. 提案書での引用ガイド

提案書からサービスカタログを引用する際の推奨パターン:

### 差別化セクション（提案コンセプト、要件対応方針）

```
当社の「[サービス名]（SVC-XXX）」は[差別化ポイント]を特長とし、
[対象課題]に対して[期待効果]を実現します。
[業種]における[規模]のお客様で[効果]の実績があります。
```

### 体制セクション（開発体制、運用体制）

```
[サービス名]の専門チームがプロジェクトに参画し、
[フェーズ]において[役割]を担います。
```

### 実績セクション（導入実績）

```
[サービス名]は過去[N]年間で[件数]件の導入実績があり、
[業種]を中心に[効果]を達成しています。
```
