## 3. 引用ガイドライン

### 提案書での引用方法

```
当社のホワイトペーパー「[タイトル]」（[発行年月]）に基づき、
[技術/方法論] を [対象領域] に適用します。
```

### 引用時の注意

- ホワイトペーパーの内容を提案書にコピーする場合は要約・再構成すること
- 原典との整合性を保つため、数値データはホワイトペーパーの最新版を参照すること
- クライアント固有の文脈に合わせて記述を調整すること（汎用的な記述をそのまま転記しない）

## ホワイトペーパー追加手順

1. `whitepapers/` ディレクトリに Markdown（推奨）または PDF を追加
2. 本ファイル（`index.md`）の一覧テーブルにエントリを追加
3. タグを付与し、引用推奨マップを更新
4. 最終更新日を更新
