# タスクトラッカー

提案策定中に発生したタスク・検討事項・課題を一元管理するファイル。

## 書き方

**最小限これだけでOK:**

```
### TASK-{連番}: タイトル

説明を一行書く
```

省略されたフィールドは Claude が補完します（優先度→medium、カテゴリ→内容から推定、起票日→当日）。

**しっかり書く場合:**

```
### TASK-{連番}: タイトル

- **優先度**: high | medium | low
- **カテゴリ**: 検討事項 | TODO | 質問 | 課題 | 決定事項

説明を書く
```

> **Tips**: `- **ステータス**`, `- **起票元**`, `- **起票日**`, `- **関連**` は書かなくて大丈夫です。Claude / `/task-add` が自動で埋めます。手で書きたい場合はどのフィールドでも自由に追加できます。
