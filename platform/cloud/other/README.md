## Azure 参考マッピング

AWS/GCP テンプレートの各コンポーネントを Azure に読み替える場合の対応表:

| 機能カテゴリ | AWS | GCP | Azure |

| **ネットワーク** | VPC | VPC | VNet |
| **プライベート接続** | PrivateLink | Private Service Connect | Private Endpoint |

| **IAM** | IAM Role | Service Account | Managed Identity |
| **暗号化キー管理** | KMS | Cloud KMS | Key Vault |
| **監視** | CloudWatch | Cloud Monitoring | Azure Monitor |
| **ログ管理** | CloudTrail | Cloud Audit Logs | Activity Log |
| **IaC** | CloudFormation | Deployment Manager | ARM / Bicep |

## カスタマイズ手順

1. `aws/README.md` または `gcp/README.md` をベースにコピー
2. コンポーネント名を対象プロバイダーに読み替え
3. プロバイダー固有の制約事項を追記
4. `architecture-policy.md` のインフラ構成セクションを更新
